# Cathedral Network – UNDP Crisis Mapping Demo

## Overview
This is the Cathedral Network submission for the UNDP Crisis Mapping Challenge. It is an offline-first, multi-lingual damage assessment tool that allows communities to report infrastructure damage, classify it, and geolocate it on a map.

## Features
- **Offline-first**: Works without internet, syncs when online.
- **10 languages**: English, French, Arabic, Spanish, Kiswahili, Portuguese, Chinese, Russian, Farsi, Ukrainian.
- **Map with satellite layer**: Esri satellite imagery with building footprints.
- **AI-assisted damage classification**: Zero-shot CLIP model.
- **Export data**: JSON, CSV, GeoJSON.
- **HEWD dashboard**: Humanitarian Early Warning Dashboard with compound risk score.
- **Community impact**: Leaderboard and impact metrics.
